# loads
最精巧封装，按顺序同步或异步加载js、css文件

```js
// 设置最后一个参数为'async',加载资源为异步加载，数组中的资源加载不分先后，全部加载下来，再执行主方法的回调函数
// 缺省值为同步加载，数组中的资源是按顺序加载的，第一个加载完，才会加载第二个，直到加载完所有，再执行主方法的回调函数
// 因为资源相互之间可能会产生依赖，所以保留了这种方式，弊端是，加载和传统方式一样，速度没有异步的更快。

loads(['css!http://172.31.0.115:8020/css/index.css','js!http://172.31.0.115:8020/js/index.js'],function(){
    console.log('加载完成')
},'async'); // 设置最后一个参数为'async',加载资源为异步加载，缺省值为同步加载
```
